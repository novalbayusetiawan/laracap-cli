# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-12-10

### Added
- Initial release
- Command line interface for LaraCap deployment
- Configuration management with `config:server-url` command
- Authentication with `login` and `logout` commands
- Bundle creation and upload with `apps:bundles:create` command
- Support for selecting target application
- Automatic ZIP file creation of specified directory
- Progress indicators for long-running operations
- Error handling and debug file retention
